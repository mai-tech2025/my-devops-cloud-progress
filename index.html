About Our Hangout Program

The hangout Is For LionTech DevOps and Cloud Engineers to fellowship together while adding value in a comprehensive 3 days workshop intensive training designed to transform beginners into job-ready cloud professionals. Our curriculum covers:

  • Linux System Administration
  • Python and Bash Shell Scripting
  • Resume and Linkedin Review
  • Github Actions
  • Networking Fundamentals
  • Containerization with Docker
  • Orchestration with Kubernetes
  • Grafana and slacks
  • Resume and linkedin reviews
  • python and bash Scripting
  • Cloud Platforms (AWS, Azure, GCP)
  • CI/CD Pipelines
  • Infrastructure as Code (Terraform)
  • Monitoring and Logging

Our hands-on approach ensures students work on real-world projects and graduate with a professional portfolio.
